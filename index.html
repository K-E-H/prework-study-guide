HTML

  • The head element contains information about the webpage.
  • The title element in the head section is visible in the browser's title bar.
  • The body element represents the visible content shown to the user.
  • The body element is a semantic element, it's name describes it's content and role.
  • Non-semantic elements like div and span can be used to create page and line divisions.
  • HTML elements provide structure for the content.
  • Elements nested inside other elements are called child elements
  • Visible HTML elements need to have an opening and a closing tag

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds all modifications in the current working branch to the staging area.
  • git commit -m "message here to describe what commit contains": updates the branch.
  • git pull origin main: should be used to make sure you have the latest version locally before you start working on the code.
  • git push origin feature/branch-name: pushes local changes to the remote branch.
